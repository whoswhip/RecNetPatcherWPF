# RecNet Patcher WPF
This is a GUI version of [RecNetPatcher](https://github.com/whoswhip/RecNetPatcher) made in WPF.

## Building
**Requirements**:
- .net 10 SDK
- [RecNetPatcher](https://github.com/whoswhip/RecNetPatcher) in the same root directory as RecNetPatcherWPF
    - NOT in the same directory, just the same directory RecNetPatcherWPF is in

**Building**:
You can build using `dotnet build -c Release` or using Visual Studio.
